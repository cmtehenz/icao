# Squawk

**Catalog ID:** 0139  
**Category:** ATC Phraseology  
**Priority:** Critical  
**ICAO Parts:** Part 2

## ✈ Mission Brief

Today's lesson covers Squawk — the ATC instruction to set a specific four-digit transponder code.

Squawk is the everyday transponder code assignment.

This lesson is distinct from Squawk Ident (0106), which covers the IDENT button function. Squawk is about setting the correct discrete code that makes your aircraft identifiable and traceable on secondary surveillance radar.

In the H130, operating in Brazilian controlled airspace — Congonhas, Florianópolis, Curitiba, Joinville — you will be assigned a squawk code by ATC before or shortly after departure.

Understanding the instruction, setting the code correctly, and reading it back are non-negotiable precision skills.

## 📖 Meaning

Squawk is an ATC instruction directing the pilot to set a specified four-digit octal code on the aircraft's transponder for secondary surveillance radar identification and tracking.

## 🇧🇷 Portuguese Meaning

Transponha / Sintonize o transponder no código.

Instrução do controle para o piloto inserir um código de quatro dígitos no transponder da aeronave, permitindo sua identificação e rastreamento no radar secundário de vigilância.

## 🛩 Operational Meaning

When Squawk is received, the pilot should:

- enter the four-digit code on the transponder immediately
- verify the code is set correctly before acknowledging
- read back the full four-digit code
- confirm the transponder is in ALT mode after code entry
- not change the code without further ATC instruction

Common Squawk code assignments include:

- discrete codes assigned per flight for individual tracking
- Squawk 2000 — VFR or uncontrolled airspace (ICAO standard)
- Squawk 7700 — Emergency
- Squawk 7600 — Radio failure
- Squawk 7500 — Unlawful interference (hijack)
- Squawk 2200 — Standard IFR code in some Brazilian sectors

**Important:** The code 7500, 7600, and 7700 are reserved emergency codes. Never squawk these accidentally. If you select a code and approach these numbers, slow down and verify each digit. The consequences of an inadvertent emergency squawk are immediate.

## 🎯 Why ATC Uses It

ATC assigns discrete squawk codes to:

- individually identify each aircraft on radar
- track position, altitude, and routing
- distinguish conflicting radar returns
- coordinate between sectors during handoffs
- enable Mode C altitude readout alongside the code

## 🌍 Real Operational Context

You are taxiing at Congonhas for departure in your H130.

Clearance Delivery transmits:

ANAC123, cleared to Florianópolis as filed, flight level eight zero, squawk two four zero five.

You enter 2405 on the transponder, verify each digit, then transmit:

Cleared Florianópolis as filed, flight level eight zero, squawking two four zero five, ANAC123.

After takeoff, São Paulo Approach confirms:

ANAC123, radar contact, squawk confirmed, two miles south of Congonhas, climb flight level eight zero.

Your discrete code is now your radar identity for the entire flight.

## 📡 Real ATC Phraseology

1. Delivery: ANAC123, squawk two four zero five.
2. Approach: ANAC123, squawk three three six zero.
3. Control: ANAC123, squawk four five one two, climb flight level one zero zero.
4. Approach: ANAC123, re-squawk two one zero four, confirm when set.
5. Control: ANAC123, squawk two zero zero zero, leaving controlled airspace.

## 👨‍✈️ Captain Delta Teaching

Squawk is a precision task.

Four digits. Each one matters.

A single wrong digit creates a conflict — your radar return belongs to another squawk code, another flight, possibly another sector.

Brazilian pilots sometimes rush the code entry during a busy clearance delivery phase.

Slow down. Enter. Verify. Then readback.

The readback is your confirmation, but it only works if you verify the panel before you speak.

Common trap: ATC says squawk two four zero five. You enter two four five zero. You read back the correct code from memory. ATC hears the correct readback. But your transponder shows the wrong code.

The error does not surface until radar contact — and by then, you are identified under someone else's squawk.

Enter. Verify. Readback.

In the H130, the transponder is panel-mounted. Make code entry part of your before-takeoff flow.

## ❓ Common ICAO Speaking Question

What do you do when ATC assigns a new squawk code during flight?

**Good ICAO Level 4 Answer**

When ATC assigns a new squawk code, I enter the four-digit code on the transponder, verify each digit is correct, then read back the code: Re-squawking [code], ANAC123. I confirm the transponder remains in ALT mode. I do not change the code without ATC instruction, and I am careful to avoid accidentally selecting 7500, 7600, or 7700 when dialing through the code range.

## 🇧🇷 Common Brazilian Mistakes

- ❌ Reading back the code from memory without verifying the transponder display.  
  ✔ Enter, verify on the panel, then read back.

- ❌ Leaving the transponder in STANDBY after entering the new code.  
  ✔ Ensure ALT mode is selected so Mode C altitude is transmitted.

- ❌ Abbreviating the readback — saying "squawk set" instead of the four digits.  
  ✔ Always read back the full code: Squawking two four zero five, ANAC123.

- ❌ Confusing Squawk with Squawk Ident — entering a code when ident was requested.  
  ✔ Squawk = set a code. Squawk Ident = press the IDENT button. Two different actions.

## 🗣 Pronunciation Coaching

**Target Phrase:** Squawking two four zero five, ANAC123

**Pronunciation:** SKWOK-ing TOO FOR ZE-ro FIFE

**Word Stress**

- Squawking → SKWOK-ing
- Zero → ZE-ro (not "oh" — in ICAO, say "zero")
- Five → FIFE (ICAO standard for digit five)

Practice:

Two... Four... Zero... Five...

Together:

Squawking two four zero five, ANAC123.

## 🎯 Captain Challenge

Curitiba Clearance Delivery issues your full IFR clearance for Navegantes and assigns: "squawk three five two one."

**Your first task**

Write out the complete readback including the squawk code. What happens to your transponder?

**Your second task**

Airborne, Curitiba Control says: "ANAC123, re-squawk four four one zero." What do you say and do?

**Expected ICAO Level 4 Answer**

In the readback I include all clearance elements and end with: squawking three five two one, ANAC123. I enter 3521 on the transponder, verify each digit, and ensure ALT mode is selected. When airborne and instructed to re-squawk 4410, I enter the new code, verify it, and transmit: Re-squawking four four one zero, ANAC123. I do not confirm until the code is set and verified on the panel.
