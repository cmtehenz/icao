# Report Base

**Catalog ID:** 0023  
**Category:** ATC Phraseology  
**Priority:** High  
**ICAO Parts:** Part 2

## ✈ Mission Brief

Today's lesson covers the report that connects downwind and final.

When Tower says Report Base, they want to know when you turn onto the base leg — the crosswind portion of the pattern that sets up your final approach.

This report helps ATC confirm spacing and decide when to issue landing clearance or further instructions.

## 📖 Meaning

Report Base instructs the pilot to advise ATC when the aircraft is on the base leg of the traffic pattern for the assigned runway.

## 🇧🇷 Portuguese Meaning

Reporte base.

Informe ao controle quando a aeronave estiver na perna base do circuito de tráfego.

## 🛩 Operational Meaning

When ATC issues Report Base, the pilot should:

- continue the pattern as cleared
- report when established on the base leg
- specify left or right base when applicable
- include the runway and aircraft identification
- remain prepared for final instructions or landing clearance

Base reports are common:

- during VFR tower operations
- when sequencing traffic from downwind to final
- when Tower needs to confirm spacing before clearing an aircraft to land
- in busy circuits with multiple aircraft

## 🎯 Why ATC Uses It

Controllers use Report Base to:

- confirm the aircraft is turning from downwind toward final
- manage spacing between aircraft on base and final
- issue timely landing clearances
- detect early or late turns that affect separation
- maintain an accurate traffic picture

## 🌍 Real Operational Context

You are on left downwind for runway one eight at Navegantes in your H130.

Tower instructs:

Tower: ANAC123, report base runway one eight.

You continue downwind until abeam the normal turn point, then turn base.

When established on base, you report:

Pilot: Base runway one eight, ANAC123.

Tower may then clear you to continue or instruct you to report final.

## 📡 Real ATC Phraseology

1. Tower: ANAC123, report base runway one eight.
2. Tower: ANAC123, report base.
3. Tower: ANAC123, report left base runway two seven.
4. Tower: ANAC123, extend downwind, report base.
5. Tower: ANAC123, report base, number two following traffic on final.

## 🎙 Professional Pilot Readbacks

1. Wilco, ANAC123.
2. Will report base runway one eight, ANAC123.
3. Will report base, ANAC123.
4. Base runway one eight, ANAC123.
5. Left base runway two seven, ANAC123.

## 👨‍✈️ Captain Delta Teaching

If downwind is the long parallel leg, base is the turn that points your nose toward the runway.

It's the bridge between "I'm in the pattern" and "I'm on final."

When ATC asks for Report Base, they want to know you've made that bridge.

Why?

Because base is where spacing problems often appear.

If you turn base too early or too late, the aircraft ahead or behind you may no longer be safely separated.

A timely base report gives Tower time to react — hold you on downwind, extend, or clear you to land.

## ❓ Common ICAO Speaking Question

Why is the base report important for traffic separation?

**Good ICAO Level 4 Answer**

The base report helps ATC confirm when an aircraft is turning toward final approach. This allows the controller to manage spacing between aircraft and issue landing clearance or other instructions at the correct time.

## 🇧🇷 Common Brazilian Mistakes

- ❌ Reporting base while still on downwind.  
  ✔ Report only when established on the base leg.

- ❌ Turning base without spacing awareness.  
  ✔ Maintain visual scan and follow Tower sequencing instructions.

- ❌ Omitting the runway in the report.  
  ✔ Say: Base runway one eight, ANAC123.

- ❌ Assuming base report means cleared to land.  
  ✔ Landing clearance is still required unless already issued.

## 🗣 Pronunciation Coaching

**Target Phrase:** Report Base

**Pronunciation:** ri-PORT bays

**Word Stress**

- Report → second syllable
- Base → one syllable, clear "bays" sound

Practice:

Report... Base...

Together:

Report Base.

Then inside a complete transmission:

Base runway one eight, ANAC123.

Do not rush the word base — keep it distinct from "race."

## 🧠 Memory Trick

**B**ase = **B**ridge to final.

You report when you cross the bridge from downwind toward the runway centerline.

## 🎯 Captain Challenge

ATC says:

Tower: ANAC123, extend downwind, report base runway one eight.

**Your first task**

Read back exactly as a professional pilot.

**Your second task**

Answer this question:

Tower asked you to extend downwind before reporting base. What does that mean operationally?

**Expected ICAO Level 4 Answer**

I should continue on downwind longer than normal before turning base. This is usually for traffic spacing. I must not turn base until spacing is appropriate and I am ready to report base as instructed.
