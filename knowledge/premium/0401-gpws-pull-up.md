# GPWS Pull Up

**Catalog ID:** 0401  
**Category:** Emergency  
**Priority:** Critical  
**ICAO Parts:** Part 2 · Part 3

## ✈ Mission Brief

Today's lesson covers GPWS Pull Up — the highest-priority Ground Proximity Warning System command requiring an immediate maximum-performance climb to avoid terrain impact.

"Pull up" is not advisory. It is an escape order.

In the H130, a terrain pull-up means collective up, maximum power, climb attitude, and an immediate Mayday or urgency call once the aircraft is climbing. Brazilian mountainous and coastal escarpment operations leave no margin for hesitation.

## 📖 Meaning

GPWS Pull Up is the critical aural command from the Ground Proximity Warning System directing the pilot to execute an immediate maximum-performance climb because terrain impact is imminent or the closure rate with terrain is critically unsafe.

## 🇧🇷 Portuguese Meaning

GPWS Pull Up / suba imediatamente.

Comando sonoro crítico do sistema de aviso de proximidade com o solo determinando que o piloto execute imediatamente uma subida de máximo desempenho porque o impacto com o terreno é iminente ou a taxa de fechamento com o relevo é criticamente insegura.

## 🛩 Operational Meaning

When GPWS commands Pull Up, the pilot should:

- apply maximum power / collective immediately
- pitch to the pull-up attitude without delay
- do not delay for configuration changes that reduce climb
- climb until the warning stops and terrain clearance is assured
- advise ATC: GPWS pull up, climbing, mayday if required

GPWS Pull Up reporting commonly includes:

- mayday, GPWS pull up, climbing
- GPWS pull up, request maximum altitude
- clear of terrain, cancel mayday if applicable
- request vectors away from terrain

**Important:** Pull up first. Configure later. Never descend to "check" a pull-up warning.

## 🎯 Why ATC Uses This Concept

Controllers respond to GPWS Pull Up reports to:

- clear all conflicting altitude assignments immediately
- provide unrestricted climb when possible
- turn aircraft away from known high terrain
- alert the sector and adjacent units
- coordinate emergency handling and possible diversion

## 🌍 Real Operational Context

You are on a night approach toward Belo Horizonte in your H130. Suddenly: "TERRAIN, PULL UP."

You apply maximum collective and climb.

You transmit:

Belo Horizonte Approach, ANAC123, mayday mayday mayday, GPWS pull up, climbing, request five thousand.

Approach clears unrestricted climb and turns you north away from the ridge.

When the warning ceases and you are stable, you update intentions.

## 👨‍✈️ Captain Delta Teaching

Pull up means the ground is winning the race.

There is no soft version of this response. Maximum performance climb. Now.

In jets, that means fire wall thrust and pull to the stick-shaker edge. In the H130, it means maximum collective within rotor limits, climb attitude, and no descent until clear.

The radio call should include pull up and climbing. If terrain impact was imminent, Mayday is appropriate. Do not be shy about the distress call — ATC needs the priority.

Serra do Mar, Confins arrivals, night offshore returns to the coast: these are pull-up environments. Brief the escape before you need it.

## ❓ Common ICAO Speaking Question

You hear "Pull up" from GPWS. What do you do and what do you say?

**Good ICAO Level 4 Answer**

I would execute an immediate maximum-performance climb without delaying for analysis. I would transmit: ANAC123, mayday mayday mayday, GPWS pull up, climbing, request five thousand. I would continue climbing until the warning stops and terrain clearance is assured, then advise ATC that I am clear of terrain and state intentions.

## 🇧🇷 Common Brazilian Mistakes

- ❌ Reducing climb to troubleshoot the system mid-warning.  
  ✔ Climb until clear. Troubleshoot later.

- ❌ Using only "going around" without saying pull up.  
  ✔ Say GPWS pull up — it communicates terrain emergency.

- ❌ Waiting for ATC climb clearance before pulling up.  
  ✔ Climb first. Declare. Clearance follows.

- ❌ Cancelling mayday before terrain clearance is certain.  
  ✔ Cancel only when stable, clear, and intentional.
