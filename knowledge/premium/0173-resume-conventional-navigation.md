# Resume Conventional Navigation

**Catalog ID:** 0173  
**Category:** Navigation  
**Priority:** High  
**ICAO Parts:** Part 2

## ✈ Mission Brief

Today's lesson covers Resume Conventional Navigation — the ATC instruction that specifically directs a pilot to stop using RNAV or GPS-based navigation and revert to conventional ground-based navaid navigation such as VOR, NDB, or DME.

Resume Conventional Navigation and Resume Own Navigation (lesson 0011) are related but distinct.

Resume Own Navigation ends radar vectoring. The pilot resumes following their cleared route using any navigation means — GPS, RNAV, VOR, or FMS.

Resume Conventional Navigation is a specific navigation type instruction. It directs the pilot to abandon GPS or RNAV guidance and revert to conventional navaid tracking — VOR radials, NDB bearings, or DME arcs.

Why would ATC issue this? GPS interference. Solar event affecting GPS accuracy. RNAV procedure not approved in the sector. GPS signal anomaly. Navigation system failure on board. Or the published procedure specifies conventional navigation.

In Brazil, GPS interference events are recorded periodically near certain areas, and understanding when and how to revert to conventional navigation is an important ICAO operational skill.

## 📖 Meaning

Resume Conventional Navigation is an ATC instruction directing the pilot to discontinue RNAV or GPS-based navigation and revert to navigation using conventional ground-based navaids — VOR, NDB, or DME — for the continuation of the flight or procedure.

## 🇧🇷 Portuguese Meaning

## 🛩 Operational Meaning

When ATC instructs Resume Conventional Navigation, the pilot should:

- disengage RNAV or GPS navigation mode
- tune and identify the appropriate VOR, NDB, or DME navaid
- confirm the navaid identifier by Morse code tone
- navigate using the conventional instrument indication
- advise ATC if unable to comply due to equipment limitations
- continue to comply with all altitude, speed, and other clearance elements

The transition from RNAV to conventional navigation requires:

- selecting the correct navaid frequency
- verifying station identification
- setting the correct course on the OBS (for VOR) or bearing selector (for ADF/NDB)
- cross-checking position with DME where available

**Important:** Resume Conventional Navigation is NOT the same as Resume Own Navigation. Resume Own Navigation ends radar vectoring — the navigation type remains the pilot's choice. Resume Conventional Navigation is a specific constraint on navigation method.

## 🎯 Why ATC Uses It

Controllers issue Resume Conventional Navigation when:

- GPS interference is detected in the area
- RNAV procedures are not approved or authorized in the sector
- the aircraft GPS or RNAV system is reported unreliable
- the published procedure requires conventional navaid tracking
- air traffic management requires conventional navigation for sequencing
- navigation anomalies are detected from radar track comparison

## 🌍 Real Operational Context

You are flying an RNAV route between Florianópolis and Curitiba in your H130.

Curitiba Approach transmits:

ANAC123, GPS interference reported in the area, resume conventional navigation, proceed to CWB VOR, maintain five thousand feet.

You respond:

Resume conventional navigation, proceeding to CWB VOR, maintaining five thousand feet, ANAC123.

You disengage the GPS navigation mode.

You tune NAV 1 to CWB VOR frequency 113.3.

You listen for the CWB Morse code identifier — confirmed.

You center the CDI and track directly to CWB VOR.

Your H130 now navigates conventionally, independent of any GPS signal.

## 👨‍✈️ Captain Delta Teaching

GPS-dependent navigation has become the default for most Brazilian pilots.

That is fine — when GPS works.

But GPS is not infallible.

Solar events, ground-based interference, and equipment anomalies can degrade or eliminate GPS accuracy with little warning.

The pilot who cannot revert to VOR tracking when GPS fails is vulnerable.

Resume Conventional Navigation is ATC telling you: GPS is compromised. Use the ground-based tools.

In the H130, you have VOR navigation available. Know how to use it.

Know the difference from Resume Own Navigation:

Resume Own Navigation ends vectoring — navigate by any means.

Resume Conventional Navigation ends RNAV/GPS — navigate by VOR, NDB, or DME only.

Two different phrases. Two different situations. Precise vocabulary protects safe operations.

## ❓ Common ICAO Speaking Question

How does Resume Conventional Navigation differ from Resume Own Navigation, and when would each be issued?

**Good ICAO Level 4 Answer**

Resume Own Navigation is issued after radar vectoring ends. The controller releases the pilot to navigate by their preferred means — GPS, RNAV, FMS, or VOR — along the cleared route. The navigation method is not restricted. Resume Conventional Navigation, by contrast, specifically restricts the navigation method. The controller is directing the pilot to stop using RNAV or GPS and revert to conventional ground-based navaid navigation using VOR, NDB, or DME. This instruction is typically issued when GPS interference is reported, when RNAV is not approved in the sector, or when a published procedure requires conventional navigation. The key distinction is that Resume Conventional Navigation constrains the navigation type, while Resume Own Navigation only ends vectoring.

## 🇧🇷 Common Brazilian Mistakes

- ❌ Treating Resume Conventional Navigation as equivalent to Resume Own Navigation.  
  ✔ Resume Conventional Navigation specifically requires reverting to VOR, NDB, or DME — not GPS or RNAV.

- ❌ Continuing to use GPS after receiving Resume Conventional Navigation.  
  ✔ Disengage RNAV/GPS mode and navigate using conventional navaids only.

- ❌ Not identifying the navaid before tracking it.  
  ✔ Always listen for and confirm the Morse code identifier before navigating.

- ❌ Readback that omits the navaid: "Resuming conventional navigation, ANAC123."  
  ✔ Include the navaid: resuming conventional navigation, CWB VOR, ANAC123.

## 🎯 Captain Challenge

You are tracking an RNAV route at five thousand feet between Florianópolis and Curitiba.

Curitiba Approach transmits: ANAC123, GPS signal unreliable in area, resume conventional navigation, proceed to CWB VOR radial zero nine zero, maintain five thousand feet, QNH one zero one zero.

**Your first task**

Read back the clearance completely.

**Your second task**

Describe the cockpit sequence you follow to comply with the instruction, from disengaging RNAV to tracking the VOR radial.

**Expected ICAO Level 4 Answer**

Readback: ANAC123, resuming conventional navigation, CWB VOR radial zero nine zero, maintaining five thousand feet, QNH one zero one zero. Cockpit sequence: I disengage the GPS or RNAV navigation mode. I tune NAV 1 to CWB VOR on 113.3. I listen for the CWB Morse code identifier and confirm it. I set the OBS to 090 degrees — the VOR zero nine zero radial. I track the VOR inbound toward CWB, keeping the CDI centered, maintaining five thousand feet on QNH one zero one zero. I update my altimeter subscale if not already set to one zero one zero. I monitor the VOR indication and report any inability to identify or track the station.
