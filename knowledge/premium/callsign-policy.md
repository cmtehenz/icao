# ICAO Delta — Callsign Policy

**Applies to:** all premium Word Mission lessons.

## Part 2 (Radiotelephony)

**Default training callsign:** `ANAC123`

**Additional aircraft (multi-traffic scenarios):**

- `ANAC456`
- `ANAC789`

## Part 3 (Speaking)

Do **not** use `ANAC123` unless the exercise explicitly requires radio communication.

## Word Mission

Use `ANAC123` **only** when teaching ATC phraseology.

## Captain Delta

Use `ANAC123` **only** inside radio transmissions.

Narration, explanations, analogies, and ICAO speaking answers should **not** include a callsign.

## Where callsigns appear

| Section | Callsign? |
|---------|-----------|
| 📡 Real ATC Phraseology | Yes — `ANAC123` (or `ANAC456` / `ANAC789`) |
| 🎙 Professional Pilot Readbacks | Yes |
| 🎯 Captain Challenge (radio task) | Yes |
| 🌍 Real Operational Context | No (unless quoting a transmission) |
| 👨‍✈️ Captain Delta Teaching | No |
| ❓ ICAO Speaking answers | No |
| 🗣 Pronunciation Coaching | Only inside example transmissions |

## Example

**Correct**

> ANAC123, continue approach.

**Incorrect (Captain narration)**

> Gustavo, ANAC123, imagine you're approaching Florianópolis...

**Correct (Captain narration)**

> Gustavo, imagine you're approaching Florianópolis. Tower says: "ANAC123, continue approach."
