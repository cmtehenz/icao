# Knowledge Review Mode

Enable developer review in the app:

```bash
NEXT_PUBLIC_KNOWLEDGE_REVIEW=true
```

When active, Word Mission displays for each lesson:

- Knowledge source (primary reference)
- Entry version
- Whether fallback was used
- Curated vs legacy content
- Reference count

Use this panel while expanding the vocabulary database to catch weak entries, missing fields, and forbidden meta phrases before students see them.
