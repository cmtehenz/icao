# Philosophy

Visual Coaching exists to reduce thinking effort.

It does not teach.

Captain Delta teaches.

Visual Coaching simply helps the student focus on the right information at the right time.

The objective is reducing cognitive load.

Never adding more information.

Students should always look at less.

Think more.

Speak naturally.

---

# Mission

Visual Coaching transforms complex aviation communication into simple visual cues.

Instead of reading paragraphs,

students recognize patterns.

Ideas.

Relationships.

The objective is remembering concepts.

Not memorizing text.

---

# One Visual Rule

Captain never highlights everything.

Only one learning objective may be highlighted at any moment.

One keyword.

One pronunciation stress.

One connector.

One mistake.

One operational concept.

One visual lesson.

# Cognitive Load

Visual Coaching should always reduce information.

Never increase it.

Good examples

✓ One keyword.

✓ One arrow.

✓ One highlighted phrase.

Poor examples

✗ Long paragraphs.

✗ Many colors.

✗ Multiple highlights.

✗ Complex diagrams.

Less is always better.

---

# Captain Delta Behavior

Captain introduces visuals naturally.

Examples

"Focus only on this word."

---

"Forget the paragraph."

---

"Remember these four ideas."

---

"Look at the symbols."

Captain never explains the entire screen.

The visuals do that.

# Product Rules

Visuals should never become decorations.

Every visual must reduce cognitive effort.

Visual Coaching never replaces conversation.

It only supports communication.

Visual support must always decrease over time.
